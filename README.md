# Task 4: Logistic Regression (Breast Cancer Dataset)

## Objective
To build a binary classifier using Logistic Regression.

## Steps Performed
- Data preprocessing (handled missing values, encoded target)
- Split dataset into training and testing sets
- Applied Logistic Regression model
- Evaluated using Confusion Matrix, Precision, Recall, ROC-AUC
- Plotted ROC Curve

## Tools Used
Python, Pandas, Scikit-learn, Matplotlib
